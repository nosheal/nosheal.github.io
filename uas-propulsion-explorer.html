How to read this — triangulation & evidence

Each dot is one product. Shape encodes technology (see the "i" button next to the Technology selector for definitions). Color encodes sourcing region. Border thickness encodes how many independent sources verify the row: hairline = 1, medium = 2, heavy = 3+.

  • datasheet — vendor datasheet URL was directly verified at scrape time.
  • FCC brand — manufacturer holds at least one FCC grantee code (US-marketed RF gear).
  • FCC equipment — a specific FCC ID was matched, with grant-level frequency band & output power.
  • FAA DOC — manufacturer has an accepted FAA UAS Declaration of Compliance (Part 89 Remote ID or Part 107 OOP).

Evidence tier in detail panel: priced_catalog = public price + datasheet; b2b_no_price = RFQ-only; manufacturer_status = company-status note (not a product).

Caveat: only ~10% of rows have ≥ 2 independent sources. Treat single-source rows as preliminary. Drone-adjacent items (e.g., maritime fuel cells, aviation engines) are flagged with an orange tag and kept for completeness rather than removed.
